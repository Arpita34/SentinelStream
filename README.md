# Full-Stack Authentication System

A complete authentication system with JWT and role-based access control for a video management platform.

## Quick Start

### 1. Start Backend
```bash
cd backend
npm run dev
```

### 2. Start Frontend (new terminal)
```bash
cd frontend
npm run dev
```

### 3. Open Browser
Navigate to: http://localhost:5173
